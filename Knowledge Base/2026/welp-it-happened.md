# Welp, it happened

_I got laid-off_ from Binance.US It was a great run and such a great experience. \
\
There were so many projects, creative freedom, and ability to grow in the company from many different departments. When people say you have to wear multiple hats in this industry I definitely felt it in this role, I definitely grew a few more whites into my hair.

My role there covered Basic T2 Analyst Triage, Threat Hunting, Incident Responding on both endpoint and in AWS, Threat detection and fine tuning alerts. And that was just the job description. My Director believed in my capabilities and built out a Threat Intelligence Pipeline consisting of a MISP Server > Jira Webhook > AWX Ansible Server (automation) > finally to our Crowdstrike via API. It was definitely a head banger of an automation project but it was done.\
\
What I learned during that engineering project was a great ordeal of a PRD (Production Requirement Document) stating the why and the how. I learned how to conduct testing to ensure that while building everything is secure. Additionally, I learned versioning and ensuring versions across all tools are adaptable with each other. And lastly SDLC - keeping up these server via AWS and ensuring these are up to date with out messing up the compatible versions is a huge task and if no snapshot redundancies are not created, we would have to create a whole new pipeline.\
\
On top of this, I'd help out the Senior Fraud Director _(former FBI)_ on the dark net to capture threat actors and to see if companies popped that held crypto would be affected. That was fun.\
\
Overall, the experience I've earned working there is invaluable and I hope to continue working for like minded individuals in the future!
